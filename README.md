
Grocery Store Management System Web Application
_______________________________________________
Welcome to the Grocery Store Management System project! This application is designed to streamline the management of grocery store operations, including inventory management, order processing, and sales reporting.

Project Overview
This project is a 3-tier web application built using the following technologies:

Front-end: HTML, CSS, JavaScript, and Bootstrap
Back-end: Python with Flask framework
Database: MySQL
Features
User-Friendly Interface: A clean and intuitive UI for easy navigation and efficient management.
Real-Time Inventory Management: Track and manage stock levels with automated alerts for low inventory.
Order Processing: Seamless order management for quick processing and accuracy.
Sales Reporting: Detailed reports on sales performance and customer purchasing trends to help make informed business decisions.

Installation
To set up this project on your local machine, follow these steps:

Clone the repository:


git clone https://github.com/your-username/Grocery_Store_Management_System_webapp.git
cd Grocery_Store_Management_System_webapp
Install the required dependencies: Make sure you have Python and pip installed, then run:

pip install mysql-connector-python
pip install flask
Configure the MySQL database:

Install MySQL and set up a new database.
Update the database configuration in the application to connect to your MySQL instance.
Run the application: Start the Flask server to run the application locally:

flask run

Access the application: Open your web browser and go to http://127.0.0.1:5000/ to view the application.

Usage
Login: Access the system by logging in with your credentials.
Dashboard: View a summary of store operations, including inventory levels and sales data.
Manage Inventory: Add, update, or delete products in your inventory.
Process Orders: View and manage customer orders efficiently.
Generate Reports: Create sales and inventory reports for detailed analysis.
